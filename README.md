A Web Application for Frequently asked question using Laravel

# EPIC NUMBER 1
Feature which allows the user to add comments for the answers posted.

# User Story
- As a user, I must be able to post the comments for an answer, so that other users can see my comments
- As a user, I need to delete the comment which I posted, so that other users cannot see the comment I deleted
- As a user, I want to view the email Id of the user who posted the comments, so that I can see who had posted the comments
- As a user, I want to view the comments just below the answer(In the answer page), so that it is easy for me to find the comments for the answers.


# EPIC NUMBER 2
Feature which allows the user to view other user questions, sorting the questions and searching the questions

# User Story
- As a user, I should be able to view the questions and answers posted by other users along with their email Id, so that I can post my views in public.
- As a user, I should be able to sort the questions I created based on recently updated, so that I can find the most recently created question by me.
- As a user, I should be able to sort the questions which everyone created based on recently updated, So that I can find the most recently created question by everyone.
- As a user, I should be able to search for the question which was created by me with the exact question name, so that I can find the question I wanted easily
- As a user, I should be able tp search for the question which was created by everyone with the exact question name, so that I can find the question I wanted to view.
- As a user, I should be able to view the questions and answers posted by me, so that I can view my contribution.



# EPIC NUMBER 3
Feature which allows the user to post the votes for the answers posted

# User Story
- As a user, I should be able to vote up for an answer, so that I feel the answer is worthy.
- As a user, I should be able to vote down for an answer, so that I feel the answer is not good.
- As a user, I should be able to view the count of the vote up and vote down, so that I can analyse other user opinions.
- As a user, I should be able to view the voting button just below the answer, so that I can vote for the answer.



# EPIC NUMBER 4
Feature allows only the owner of the questions, answers and comments to make changes to the existing content.

# User Story
- As a user, I should be the only one who can edit my question, so that others cannot edit my question.
- As a user, I should be the only one who can edit my answer, so that others cannot edit my answer.
- As a user, I should be the only one who can delete my question, so that others cannot delete my question.
- As a user, I should be the only one who can delete my answer, so that others cannot delete my question.
- As a user, I should be the only one who can delete my comment, so that others cannot delete my comment.
